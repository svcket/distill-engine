# Warp: Editorial Depth

## Description
Maximize editorial nuance, extract deep frameworks, and prioritize quality over velocity. Use for complex, multi-layered source material.

## Configuration
- **Model Intensity**: High (use reasoning-capable models).
- **Extraction Passes**: 3 (Recursive refinement of insights).
- **Synthesis Depth**: Maximum (traverse 8+ levels of the DAG).
- **Tone**: Sophisticated, Analytical, Builder-Optimized.

## Behavioral Changes
- [Refiner]: Do not strip terminology; instead, define it within the context of the framework.
- [Analyst]: Look for contradictions and tensions within the arguments.
- [Editor]: Generate long-form articles with nested sub-sections.
