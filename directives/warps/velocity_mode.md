# Warp: Velocity Mode

## Description
Prioritize speed and concise delivery. Perfect for news, rapid takeaways, and high-frequency X updates.

## Configuration
- **Model Intensity**: Standard (fast, efficient models).
- **Extraction Passes**: 1 (Single-pass extraction).
- **Synthesis Depth**: Shallow (Direct source-to-output).
- **Tone**: Direct, Punchy, Actionable.

## Behavioral Changes
- [Refiner]: Aggressively strip filler words and tangents.
- [Analyst]: Focus only on the "Top 3 lessons".
- [Editor]: Generate threads and short bulleted takeaways.
